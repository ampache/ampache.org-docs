## Official demo

Ampache official demo site provided with funding courtesy of [Linux Australia](https://linux.org.au/).
It uses the current release version of Ampache and creative commons music from archive.org.
Functionality is reduced to allow multiple users to access the account for testing.

[Stable Branch](https://demo.ampache.dev/)

[Develop Branch](https://develop.ampache.dev/)

### Accessing the demo

The demo site can be accessed using:
username "demo"
password "demodemo"

More details about what you can do in the wiki. [demo.ampache.dev](docs/clients/demo-server)
