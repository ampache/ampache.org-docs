# Understanding Ampache Preferences

**WIP**

One thing that isn't very easy to understand are all the extra options, preferences and ways of setting things up on your server

This page will explain how things work in a (hopefully) easy way to understand.

There are 3 major types of preference:

* System Preferences
* User Preferences
* Config Preferences

User and system preferences are split into 6 categories

![image](https://user-images.githubusercontent.com/1305249/205497979-c66adec9-9ebe-4231-bd90-de3f9101e3d5.png)


## System Preferences

A System Preference's are the default settings for the System and CLI actions.

These preferences are copied to a new user when they are created. (Excluding system and plugin settings)
