## Old Information

Documentation for Ampache releases that are no longer current. These pages are kept so that anyone still running an older server, or working through an upgrade, can find the instructions that applied at the time. They are not maintained against the current release, so do not follow them when setting up a new server — start from the [Installation](../installation/installation.md) and [Configuration](../configuration/configuration.md) pages instead.

* [Ampache 6 Details](./ampache6-details.md) and [Ampache 6 for Users](./ampache6-for-users.md) — what changed in the Ampache 6 release
* [Ampache 5 Changes](./ampache5-changes.md) and [Ampache 5 for Users](./ampache5-for-users.md) — what changed in the Ampache 5 release
* [Installation (v4)](./installation-v4.md) — the installation guide as it stood for Ampache 4
* [MySQL FAQ](./mysql-faq.md) — older database questions and fixes
* [Ampache Laravel Next Generation Preview](./ampache-laravel-next-generation-preview.md) — a preview of a rewrite that was not adopted

If you are upgrading from one of these releases, read the [Upgrade](../information/upgrade.md) page first.
