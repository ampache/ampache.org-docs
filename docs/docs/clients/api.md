## Tools and plugins using the API

There are a lot of possibilities with the Ampache API and it's becoming very able to perform a lot of things that weren't possible previously.

| Tool / Library / Plugin                                          | Operating Systems     | Description                                                                                                         |
|------------------------------------------------------------------|-----------------------|---------------------------------------------------------------------------------------------------------------------|
| :-------:                                                        | :-------------:       | :-------:                                                                                                           |
| [ampache-fm](https://github.com/lachlan-00/ampache-fm)           | Linux                 | A Rhythmbox plugin capable of caching locally and syncing playback with Ampache in a format compatible with last.fm |
| [ampache-scripts](https://github.com/lachlan-00/ampache-scripts) | Linux tested (Python) | A collection of scripts for managing Ampache                                                                        |
| [python3-ampache](https://github.com/lachlan-00/python3-ampache) | Linux tested (Python) | A Python3 library for integrating Ampache into your python programs                                                 |
