<div class="text--center"><br/>
## Developers
</div>

Ampache has been around for many years now and we are still here thanks to all our [contributors](https://github.com/ampache/ampache/blob/develop/docs/ACKNOWLEDGEMENTS.md).

Be part of it, that's the best donation you could do.

If Ampache is really where you want your money to go then consider helping out some of our [downstream](https://github.com/ampache/ampache?tab=readme-ov-file#credits) API devs.

<div class="display--flex">
  <div class="article--box">
    ### Plugins
    <p class="article-p">
    Some features (lyrics, catalogs, scrobbling ...) are highly independent. You can create your one without much effort and without the need to know much about Ampache code.
    </p>
    <p class="article-p">
    <a class="box--link" href="/docs/plugins/writing-plugins" title="read more">read more</a>
    </p>
  </div>
  <div class="article--box">
    ### Translation
    <p class="article-p">
    Ampache is translated to several languages. You can improve current translation or even add new ones!<br/>
    </p>
    <p class="article-p">
    <a class="box--link" href="/docs/development/TRANSLATIONS" title="read more">read more</a>
    </p>
  </div>
  <div class="article--box">
    ### General Development
    <p class="article-p">
    Improve Ampache with new features and bug fixes. We can create a better streaming solution together, you just need to know PHP / JavaScript / HTML5.<br/>
    </p>
    <p class="article-p">
    <a class="box--link" href="/docs/development/CONTRIBUTING" title="read more">read more</a>
    </p>
  </div>
</div>

## Other ways
</div>

<div class="display--flex">
  <div class="article--box">
    ### User Support
    <p class="article-p">
    Answering questions, tests and bug isolation take us a lot of energy. That means less time for feature development. Being part of the community and helping users is an important role. Don't underestimate this work, it's an important part of Ampache's good health.
    </p>
    <p class="article-p">
    <a class="box--link" href="https://github.com/ampache/ampache/issues" title="read more">read more</a>
    </p>
  </div>
  <div class="article--box">
    ### Money
    <p class="article-p">
    The Ampache project has been running smoothly for a few years now and doesn't need to seek donations.<br/>
    Do you use one of the many Ampache API clients and subprojects? Please consider donating to them instead!
    </p>
    <p class="article-p">
    <a class="box--link" href="https://github.com/ampache/ampache?tab=readme-ov-file#credits" title="read more">read more</a>
    </p>
  </div>
